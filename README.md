## DESCRIPCIÓN DEL HTML:
Para la construcción de las páginas HTML se utilizamos etiquetas semánticas como header, nav,
main, section, article, aside, footer. Dentro de la etiqueta header tenemos el título principal y 
una barra de navegación que nos permite movernos entre las distintas páginas.

Desarrollamos cuatro páginas:
- Principal (index.html): se presentan estadísticas generales (proyectos, tareas, mensajes) y una 
sección de novedades.
- Explorador de Proyectos (proyectos.html): contiene varios artículos, cada uno representando un
proyecto con su categoría.
- Detalle de Proyecto (detalle.html): presenta información específica de un proyecto. Incluye 
descripción, fecha, autores, recursos y equipo de trabajo.
- Perfil de Usuario (perfil.html): muestra la información del usuario.
