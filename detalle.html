Descripción

Este proyecto introduce a los estudiantes en el mundo del desarrollo web, explorando las tecnologías fundamentales que conforman una aplicación moderna. A lo largo del proyecto se trabaja con HTML para estructurar el contenido, CSS para diseñar interfaces visualmente atractivas y JavaScript para agregar interactividad y dinamismo a las páginas. Los alumnos aprenden a planificar y desarrollar proyectos reales desde cero, aplicando buenas prácticas de programación como la organización del código, el uso de etiquetas semánticas y la separación entre estructura, estilo y comportamiento. Además se fomenta el trabajo en equipo y la resolución creativa de problemas técnicos que surgen durante el desarrollo.
